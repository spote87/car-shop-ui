# Frank's Car Shop App

**How to run it:**

###### Prerequisite:
1. Node

###### **Run locally:**

    _npm install_
    
    _bower install_
    
    _grunt build_

    _grunt_
    
 Last command will launch application locally. 
 
 **Note**: If you see _connection refused_ error, please refresh the page as sometimes browser is opened before server is fully started.
